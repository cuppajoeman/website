mathlib map

motivation

When I first found out about mathlib, I found it difficult to find theorems that I was interested in, I wanted to find some elementary facts about natural numbers after completing the natural number game.

It was this theorem 
            
@[protected, simp]
theorem nat.zero_eq_mul {a b : ℕ} :
0 = a * b ↔ a = 0 ∨ b = 0

Since I had thought this had to do with number theory I looked under the number theory section in mathlib, after lots of searching I realized it wasn't there. Instead it was in data.nat.basic, and upon further discussion, I learned by talking to someone else that data is where basic facts are stored.

Due to some of the confusion I've had with mathlib, I decided to start this page where we explain the structure. I am open to people contributing to this webpage so it can be improved upon, to start modifying it, go here

naming conventions

the logic behind the naming of theorems

names of common things in mathlib

  • nat - natural numbers
  • fun - function
  • int - integers
  • rat - rationals
  • zmod - integers mod n
  • fp - floating-point numbers
  • eq - equals
