grab key

The key used to grab the window is usually alt, there are two problems with this in that the alt key is used in various prgroams to allow you to select options through a gui menu without moving your mouse, which is quite useful, so therefore I recommand you remap this key through the window manager tweaks program, personally I've remapped it to the super key

using transparent windows

before you can use this you have to go into the window manager tweaks section and deselect Use mouse wheel on title bar to roll up the window.
