Anchor Intervals

Note Intervals

Before we get started, we'll recall that we defined what the interval between two numbers is, so given two numbers X and Y, then the interval between them is the number you add to get from X to Y, namely Y - X.

Since our notes have numbers in them we can also do something similar, so given two notes X* and Y*, we define the interval between them to be Y - X, the result is a number which is not a note.

The interval between two notes could be any distance from 0 to 11 (we'll forget about negative numbers for now, since a negative is equivalent to one of the positive ones anyways). And in the standard system they have developed names for each of the different sizes of intervals, this will be useful so we understand what other musicians are talking about when the say an interval of a fourth

Anchor Intervals

In our first lesson we touched on considering notes with respect to some anchor note, for example if our anchor note was 8* (Ab) then the note 9* would be considered as 1. Whenever we specify a note like this, it will be called an anchor interval and no asterisk will be written.

More explicitly an anchor interval of a note is the interval between the anchor note and the note we're considering. We'll go into what this means throughout this article.

You might have noticed that when we set our anchor note to be 11*, that every note's anchor interval became one more than it's actual number, for example 3* was 4 and 9* was 10. This was no mere coincidence, in this lesson we will explore what's actually going on here.

For a new example, let's set our anchor note to be 2* (D) and let's consider notes with respect to that anchor note. If we consider 4*, we can see that that's 2 up from 2*, so it's interval with respect to the anchor note would be 2.

But the question we really just answered was, which number do we add to 2* to get to 4*, and that number is 2. In a different way our question was just 2* + x = 4*, which can be rearranged to be x = 4* - 2* which is equal to 2.

Due to this we have gained a method to determine the interval between any two notes, so given a note X* and a note Y*, the interval from X* to Y* is Y - X.

Now that we've discussed some of the fundamental properties about anchor intervals, we can discuss how we can determine any anchor interval. Let's start with a specific example, and then come up with the general method.

So let's say our anchor note is set as 8* and our note is 6*, then since the interval from 8* to 6* is -2 or 10, then 6*'s anchor interval is 10.

So given an anchor note of Y* and a note of X*, then we take the positive version of Y* - X* as the anchor interval.

Computing Anchor Intervals efficiently

First let's repeat the method we already know and see if we can improve it. Our current method is that we take the anchor note X* and the other note we're considering Y*. Our goal is to figure out what the anchor interval of that note is, since we know that the anchor interval is defined as the interval between the anchor note and the other note, then all we have to do is figure out what Y - X is and we have our anchor interval.

One of the components of the above process which can be easier is if we could make X as small as possible so that when we subtract it from Y we don't have to "move" very much. The way we can minimize this movement is by using certain equivalent numbers to X that are "smaller".

So for example if our anchor note was 8* and our other note was 2* then instead of computing 2 - 8 which involves a larger jump, since 8 is the same as -4 in our system we could compute 2 - (-4) = 2 + 4 = 6.

Right now we're talking about notes, but since we already know how to add and subtract notes then we can use the efficient method for figuring out intervals which is to compute I(X*, Y*) as Y - s(X). If you don't remember this then go back to "numbers in a cyclic system".

The Process

  • Take your anchor note and consider the smallest representation of that number. For example if my anchor note was 8*, then I would be thinking of the number -4.
  • Now take your other note and subtract the small representation of the anchor note from your note, the result is the anchor interval. Note that if you're subtracting a negative number you can just add it.
