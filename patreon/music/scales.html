Scales

With the major scale you might know the standard way we construct it, which is by taking some root note and then following the following pattern.

W W H W W W H

Where W stands as a Whole Tone and H stands as a Half Tone. So starting on C, we get the following notes:

C D E F G A B C

Another way to write this could be in line with our previous system so we can take our W H pattern and write it as

2 2 1 2 2 2 1

Then instead of taking jumps with respect to the previous note, we could consider them with respect to the root of the scale and get these intervals:

0 2 4 5 7 9 11 0

So if we were to make an F major scale, then we would take 5* (F) and generate the following notes from it

  • 5* + 0 = 5* (F)
  • 5* + 2 = 7* (G)
  • 5* + 4 = 9* (A)
  • 5* + 5 = 10* (Bb)
  • 5* + 7 = 0* (C)
  • 5* + 9 = 2* (D)
  • 5* + 11 = 4* (E)

If we consider the minor scale, we have this pattern

W H W W H W W

In terms of numbers, that is

2 1 2 2 1 2 2

And with respect to the root note

0 2 3 5 7 8 10 0
