x11 keyboard generator

This program lets you generate new keyboard configurations with ease just by creating layouts in textfiles.

web

permalinked website

A script which allows you to maintain a website using a hierarchy structure while also being able to access pages through permalinks that know nothing of the hierarchy structure. The point of this program is to help maintain a website or hosted data that you want to exist for a long time and always have working links.

  • status: stable

fsweb

A script which generates html to support the viewing of a regular filesystem on the web

  • status: early stage development

interactive

cube

A game which allows you to interact with a rubicks cube. The point of the program is to be used for people who are interested in least solve moves and can be extended to view various properties of the cube while solving.

  • status: stable

mandelbrot

    A mandlebrot viewer able to generate still frames of the mandlebrot set in decent times, can be extended to support other types of two dimensional fractals.
  • status: stable
